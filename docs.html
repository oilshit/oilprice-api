GET /

First time using oilprice-api? You can use this endpoint as your first test. It should returns JSON text with "Hello, world!" message.

Request

https://oilprice-api.vercel.app/

Response

{
  message: "Hello, world!"
}
GET /:blend/:period

Get the oil and gas price based on :blend and :period parameters.

Request

Let say, we want to get monthly price of Duri field (located in Indonesia). 

https://oilprice-api.vercel.app/duri/monthly
